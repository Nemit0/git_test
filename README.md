# Github Collaboration Tutorial

0. 리포지토리 클론 해서 기기로 받아오기

2. 다른 사람들은 자신의 브랜치랑 매인을 비교해서 n commit behind면은 git rebase을 통해서 업데이트 하고 1. 반복하기
